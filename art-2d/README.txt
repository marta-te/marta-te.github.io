# Placeholder Images

This directory contains 2D artwork images.

To add your artwork:
1. Add image files here (e.g., artwork1.jpg, artwork2.png)
2. Update art-2d.json with the image paths and details
3. Images will display in the 2D Art tab on the Art page

Recommended image sizes:
- Minimum width: 800px
- Aspect ratio: Any (cards will adapt)
- Format: JPG, PNG, or WebP
