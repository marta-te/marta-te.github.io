# Project Banner Images

This directory contains banner images for your projects.

To add project banners:
1. Create a banner image for each project (landscape orientation works best)
2. Add the image here (e.g., asteria-banner.jpg)
3. Update projects.json with the image path in the "bannerImage" field

Recommended specifications:
- Size: 1200x600px (2:1 aspect ratio)
- Format: JPG, PNG, or WebP
- Quality: High resolution for best display
- Content: Screenshot, artwork, or designed banner for your project
