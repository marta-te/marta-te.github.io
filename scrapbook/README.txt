# Scrapbook Images

This directory contains your scrapbook page images.

To add scrapbook pages:
1. Add 30 scrapbook page images here
2. Name them: page-01.jpg, page-02.jpg, ... page-30.jpg
3. Update scrapbook.json with metadata for each page (colors, styles, elements, mood)
4. Users can then filter and browse your scrapbook pages

Recommended image specifications:
- Size: 1000x1000px or larger
- Format: JPG or PNG
- Aspect ratio: Square or portrait works best
- File naming: page-01.jpg through page-30.jpg (zero-padded)
